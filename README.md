
	

1- Create **folder** of your project

2- Open gitbash in the root path of your project

3- Do these commands:

- `git init`
- `touch README.md`
- `git add -A`
- `git commit -am 'init'`
- `git remote add origin 'https://github.com/sama-hanaie/project.git'`
- `git push -u origin master`
- `git checkout -b newbranch`
- `git checkout master`
- `git pull`
